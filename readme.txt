use below command to set enviroment variable for project
------------------------------------------------------------
export EMAIL_ADDRESS=<email>
export EMAIL_PASSWORD=<email pwd>
[Note : please use pwd generated by app pwd https://myaccount.google.com/apppasswords,Also enable IMAP forwarding from gmail setting]
export GOOGLE_APPLICATION_CREDENTIALS=<google secret key json path >
[Note : Database stored type should be Native stored]

---------------------------------------------------------

python3 crud_api.py
